# Contributing to Rinthy

Thanks for helping improve Rinthy.

## Contributor License Agreement

Before a pull request can be merged, every contributor must sign the Rinthy Contributor License Agreement.

The CLA text is available in [CLA.md](CLA.md).

To sign it, open your pull request and comment exactly:

```text
I have read the CLA Document and I hereby sign the CLA
```

The CLA Assistant workflow will check the signature automatically.
